How to Create and Upload a Preview Image

Step 1: Find the "Create" Button

Locate and click on the "Create" button in the decor plugin menu.

Step 2: Click the "Browse" Button

Click "Browse" to select your decor file from your computer.

Step 3: Select Your Decor File

Choose a `.png` file that represents your decoration.

Step 4: Confirm the File Selection

After selecting the file, confirm it's the correct image by checking the preview.

Step 5: Don't Submit for Review

Important: Do not enter a name or press "Submit for Review" at this stage; this will not share your decor with DiscoLook.

Step 6: Take a Selection Screenshot

Use a screenshot tool to capture the preview area.

Keyboard Shortcuts:

  • Windows: Windows Key + Shift + S
  • Mac: Command + Shift + 4
  • Linux: (Varies) Shift + Print Screen

Tools like ShareX can also help with selection screenshots.
